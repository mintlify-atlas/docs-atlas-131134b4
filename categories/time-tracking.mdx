[Time-tracking software](https://en.wikipedia.org/wiki/Time-tracking_software) is a category of computer software that allows its users to record time spent on tasks or projects.

## Applications

### ActivityWatch

Automatically track how you spend time on your devices.

- **Website**: [activitywatch.net](https://activitywatch.net)
- **Source Code**: [github.com/ActivityWatch/activitywatch](https://github.com/ActivityWatch/activitywatch)
- **License**: MPL-2.0
- **Tech Stack**: Python

### Beaver Habit Tracker

- **Website**: [github.com/daya0576/beaverhabits](https://github.com/daya0576/beaverhabits)
- **Demo**: [beaverhabits.com/demo](https://beaverhabits.com/demo)
- **Source Code**: [github.com/daya0576/beaverhabits](https://github.com/daya0576/beaverhabits)
- **License**: BSD-3-Clause
- **Tech Stack**: Docker

### Traggo

Traggo is a tag-based time tracking tool. In Traggo there are no tasks, only tagged time spans.

- **Website**: [traggo.net](https://traggo.net/)
- **Source Code**: [github.com/traggo/server](https://github.com/traggo/server)
- **License**: GPL-3.0
- **Tech Stack**: Docker, Go
