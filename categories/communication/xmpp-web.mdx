[Extensible Messaging and Presence Protocol](https://en.wikipedia.org/wiki/XMPP) Web clients/interfaces.

## XMPP Web Clients

### Converse.js

XMPP chat client in your browser.

- **Source Code:** [GitHub](https://github.com/conversejs/converse.js)
- **License:** MPL-2.0
- **Tech Stack:** Javascript

### Salut à Toi

Multipurpose, multi frontend, libre and decentralized communication tool.

- **Source Code:** [repos.goffi.org](https://repos.goffi.org/libervia-backend)
- **License:** AGPL-3.0
- **Tech Stack:** Python

### Libervia

Web frontend from Salut à Toi.

- **Source Code:** [repos.goffi.org](https://repos.goffi.org/libervia-web)
- **License:** AGPL-3.0
- **Tech Stack:** Python
