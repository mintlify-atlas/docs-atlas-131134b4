[Mailing list](https://en.wikipedia.org/wiki/Mailing_list) servers and mass mailing software - one message to many recipients.

## Mailing List Management

### Mailman

Manage electronic mail discussion and e-newsletter lists.

- **Source Code:** [GitLab](https://gitlab.com/mailman/)
- **License:** GPL-3.0
- **Tech Stack:** Python

### HyperKitty

Access GNU Mailman v3 archives.

- **Demo:** [lists.mailman3.org](https://lists.mailman3.org/)
- **Source Code:** [GitLab](https://gitlab.com/mailman/hyperkitty)
- **License:** GPL-3.0
- **Tech Stack:** Python

### Postorius

Web user interface to access GNU Mailman.

- **Source Code:** [GitLab](https://gitlab.com/mailman/postorius/)
- **License:** GPL-3.0
- **Tech Stack:** Python

### Sympa

Mailing list manager.

- **Source Code:** [GitHub](https://github.com/sympa-community/sympa)
- **License:** GPL-2.0
- **Tech Stack:** Perl

### Schleuder

GPG-enabled mailing list manager with resending-capabilities.

- **Source Code:** [0xacab.org](https://0xacab.org/schleuder/schleuder/tree/master)
- **License:** GPL-3.0
- **Tech Stack:** Ruby

## Newsletter Platforms

### Listmonk

High performance, self-hosted newsletter and mailing list manager with a modern dashboard.

- **Demo:** [demo.listmonk.app](https://demo.listmonk.app/)
- **Source Code:** [GitHub](https://github.com/knadh/listmonk)
- **License:** AGPL-3.0
- **Tech Stack:** Go, Docker

### Keila

Reliable and easy-to-use newsletter tool (alternative to Mailchimp and Sendinblue).

- **Demo:** [app.keila.io](https://app.keila.io)
- **Source Code:** [GitHub](https://github.com/pentacent/keila)
- **License:** AGPL-3.0
- **Tech Stack:** Docker
