[Communication software](https://en.wikipedia.org/wiki/Communication_software) used to provide remote access to systems and exchange files and messages in text, audio and/or video formats between different computers or users, using their own custom protocols.

## Real-Time Communication Platforms

### AnyCable

Realtime server for reliable two-way communication over WebSockets, Server-sent events, etc.

- **Demo:** [demo.anycable.io](https://demo.anycable.io)
- **Source Code:** [GitHub](https://github.com/anycable/anycable)
- **License:** MIT
- **Tech Stack:** Go, Docker

### Centrifugo

Language-agnostic real-time messaging (Websocket or SockJS) server.

- **Demo:** [GitHub demos](https://github.com/centrifugal/centrifugo#demo)
- **Source Code:** [GitHub](https://github.com/centrifugal/centrifugo)

### WBO

## Notification Services

### Apprise

Send notifications to almost all of the most popular notification services available to us today such as: Telegram, Discord, Slack, Amazon SNS, Gotify, etc.

- **Source Code:** [GitHub](https://github.com/caronc/apprise)
- **License:** MIT
- **Tech Stack:** Python, Docker, deb

### Gotify

Notification server with Android and CLI clients (alternative to PushBullet).

- **Source Code:** [GitHub - Server](https://github.com/gotify/server) | [Clients](https://github.com/gotify/android)
- **License:** MIT
- **Tech Stack:** Go, Docker

### ntfy

Push notifications to phone or desktop using HTTP PUT/POST, with Android app, CLI and web app, similar to Pushover and Gotify.

- **Demo:** [ntfy.sh/app](https://ntfy.sh/app)
- **Source Code:** [GitHub](https://github.com/binwiederhier/ntfy) | [Clients](https://github.com/binwiederhier/ntfy-android)
- **License:** Apache-2.0 / GPL-2.0
- **Tech Stack:** Go, Docker, K8S

### Notifo

Multichannel notification server with support for Email, Mobile Push, Web Push, SMS, messaging and a javascript plugin.

- **Source Code:** [GitHub](https://github.com/notifo-io/notifo)
- **License:** MIT
- **Tech Stack:** C#

### Novu

Notification infrastructure for developers.

- **Source Code:** [GitHub](https://github.com/novuhq/novu/)
- **License:** MIT
- **Tech Stack:** Docker, Nodejs

### PushBits

Notification server for relaying push notifications via Matrix, similar to PushBullet and Gotify.

- **Source Code:** [GitHub](https://github.com/pushbits/server)

- **Tech Stack:** Go

## Secret Sharing

### One Time Secret

Share sensitive information securely with self-destructing links that are only viewable once.

- **Demo:** [onetimesecret.com](https://onetimesecret.com)
- **Source Code:** [GitHub](https://github.com/onetimesecret/onetimesecret)
- **License:** MIT
- **Tech Stack:** Docker, Ruby, Nodejs

### OTS

One-Time-Secret sharing platform with a symmetric 256bit AES encryption in the browser.

- **Source Code:** [GitHub](https://github.com/Luzifer/ots)
- **License:** Apache-2.0
- **Tech Stack:** Go

### Shhh

Keep secrets out of emails or chat logs, share them using secure links with passphrase and expiration dates.

- **Source Code:** [GitHub](https://github.com/smallwat3r/shhh)
- **License:** MIT
- **Tech Stack:** Python
