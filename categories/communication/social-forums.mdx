[Social Networking](https://en.wikipedia.org/wiki/Social_networking_service) and [Forum](https://en.wikipedia.org/wiki/Internet_forum) software.

## Federated Social Networks

### Mastodon

Federated microblogging server.

- **Source Code:** [GitHub](https://github.com/mastodon/mastodon) | [Clients](https://github.com/hyperupcall/awesome-mastodon)
- **License:** AGPL-3.0
- **Tech Stack:** Ruby

### Akkoma

Federated microblogging server with Mastodon, GNU social, and ActivityPub compatibility.

- **Source Code:** [akkoma.dev](https://akkoma.dev/AkkomaGang/akkoma)
- **License:** AGPL-3.0
- **Tech Stack:** Elixir, Docker

### Pleroma

Federated microblogging server, Mastodon, GNU social, & ActivityPub compatible.

- **Source Code:** [git.pleroma.social](https://git.pleroma.social/pleroma/pleroma)
- **License:** AGPL-3.0
- **Tech Stack:** Elixir

### GoToSocial

ActivityPub federated social network server implementing the Mastodon client API.

- **Source Code:** [Codeberg](https://codeberg.org/superseriousbusiness/gotosocial)
- **License:** AGPL-3.0
- **Tech Stack:** Docker, Go

### Misskey

Decentralized app-like microblogging server/SNS for the Fediverse, using the ActivityPub protocol like GNU social and Mastodon.

- **Source Code:** [GitHub](https://github.com/misskey-dev/misskey)
- **License:** AGPL-3.0
- **Tech Stack:** Nodejs, Docker

### Iceshrimp.NET

Federated microblogging server that communicates over ActivityPub.

- **Source Code:** [iceshrimp.dev](https://iceshrimp.dev/iceshrimp/iceshrimp.net)
- **License:** EUPL-1.2
- **Tech Stack:** .NET, C#, Docker

### PixelFed

Ethical photo sharing platform, powered by ActivityPub federation (alternative to Instagram).

- **Source Code:** [GitHub](https://github.com/pixelfed/pixelfed)
- **License:** AGPL-3.0
- **Tech Stack:** PHP

### Lemmy

Link aggregator for the fediverse (alternative to Reddit).

- **Source Code:** [GitHub](https://github.com/LemmyNet/lemmy)
- **License:** AGPL-3.0
- **Tech Stack:** Docker, Rust

### diaspora*

Distributed social networking server.

- **Source Code:** [GitHub](https://github.com/diaspora/diaspora)
- **License:** AGPL-3.0
- **Tech Stack:** Ruby

### Friendica

Social Communication Server.

- **Source Code:** [GitHub](https://github.com/friendica/friendica)
- **License:** AGPL-3.0
- **Tech Stack:** PHP

### Hubzilla

Decentralized identity, privacy, publishing, sharing, cloud storage, and communications/social platform.

- **Source Code:** [framagit.org](https://framagit.org/hubzilla/core)
- **License:** MIT
- **Tech Stack:** PHP

### Movim

Modern, federated social network based on XMPP, with a fully featured group-chat, subscriptions and microblogging.

- **Source Code:** [GitHub](https://github.com/movim/movim)
- **License:** AGPL-3.0
- **Tech Stack:** PHP, Docker

### Hatsu

Bridge that interacts with Fediverse on behalf of your static site.

- **Source Code:** [GitHub](https://github.com/importantimport/hatsu)
- **License:** AGPL-3.0
- **Tech Stack:** Docker, Rust

### yarn.social

Self-Hosted, Twitter™-like Decentralised micro-logging platform. No ads, no tracking, your content, your data.

- **Source Code:** [git.mills.io](https://git.mills.io/yarnsocial/yarn)
- **License:** MIT

## Comment Systems

### Isso

Lightweight commenting server written in Python and Javascript. It aims to be a drop-in replacement for Disqus.

- **Source Code:** [GitHub](https://github.com/isso-comments/isso)
- **License:** MIT
- **Tech Stack:** Python, Docker

### remark42

Lightweight and simple comment engine, which doesn't spy on users. It can be embedded into blogs, articles or any other place where readers add comments.

- **Demo:** [remark42.com/demo](https://remark42.com/demo/)
- **Source Code:** [GitHub](https://github.com/umputun/remark42)
- **License:** MIT
- **Tech Stack:** Docker, Go

### Coral

A better commenting experience from Vox Media.

- **Source Code:** [GitHub](https://github.com/coralproject/talk)
- **License:** Apache-2.0
- **Tech Stack:** Docker, Nodejs

### Chirpy

Privacy-friendly and customizable Disqus (comment system) alternate.

- **Demo:** [chirpy.dev/play](https://chirpy.dev/play)
- **Source Code:** [GitHub](https://github.com/devrsi0n/chirpy)
- **License:** AGPL-3.0
- **Tech Stack:** Docker, Nodejs

### Artalk

Comment system built in Golang, providing a lightweight and highly customizable solution for adding comments to your website.

- **Source Code:** [GitHub](https://github.com/ArtalkJS/Artalk)
- **License:** MIT
- **Tech Stack:** Go, Docker
