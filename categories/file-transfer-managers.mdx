Web-based [file managers](https://en.wikipedia.org/wiki/File_manager).

**Related:** [Groupware](/categories/groupware)

## Applications

### Apaxy

Theme built to enhance the experience of browsing web directories, using the mod_autoindex Apache module and some CSS to override the default style of a directory listing.

- **Website:** https://oupala.github.io/apaxy/
- **Source Code:** https://github.com/oupala/apaxy
- **License:** GPL-3.0
- **Platform:** Javascript

### copyparty

Portable file server with accelerated resumable uploads, deduplication, WebDAV, FTP, zeroconf, media indexer, video thumbnails, audio transcoding, and write-only folders, in a single file with no mandatory dependencies.

- **Source Code:** https://github.com/9001/copyparty
- **Demo:** https://a.ocv.me/pub/demo/
- **License:** MIT
- **Platform:** Python

### Directory Lister

Simple PHP based directory lister that lists a directory and all its sub-directories and allows you to navigate there within.

- **Website:** https://www.directorylister.com/
- **Source Code:** https://github.com/DirectoryLister/DirectoryLister
- **License:** MIT
- **Platform:** PHP, Docker

### filebrowser

Web File Browser with a Material Design web interface.

- **Website:** https://filebrowser.org/
- **Source Code:** https://github.com/filebrowser/filebrowser
- **License:** Apache-2.0
- **Platform:** Go

### FileGator

FileGator is a powerful multi-user file manager with a single page front-end.

- **Website:** https://filegator.io/
- **Demo:** https://demo.filegator.io
- **Source Code:** https://github.com/filegator/filegator
- **License:** MIT
- **Platform:** PHP, Docker

### FileRise

Web file manager with uploads, tagging, share links, gallery/table views, and an in-browser editor.

- **Source Code:** https://github.com/error311/FileRise
- **Demo:** https://github.com/error311/FileRise?tab=readme-ov-file#live-demo
- **License:** MIT
- **Platform:** Docker, PHP

### Filestash

Web file manager that lets you manage your data anywhere it is located: FTP, SFTP, WebDAV, Git, S3, Minio, Dropbox, or Google Drive.

- **Website:** https://www.filestash.app/
- **Demo:** https://demo.filestash.app/
- **Source Code:** https://github.com/mickael-kerjean/filestash
- **License:** AGPL-3.0
- **Platform:** Docker

### Gossa

Light and simple webserver for your files.

- **Source Code:** https://github.com/pldubouilh/gossa
- **License:** MIT
- **Platform:** Go

### IFM

Single script file manager.

- **Source Code:** https://github.com/misterunknown/ifm
- **License:** MIT
- **Platform:** PHP

### mikochi

Browse remote folders, upload files, delete, rename, download and stream files to VLC/mpv.

- **Source Code:** https://github.com/zer0tonin/Mikochi
- **License:** MIT
- **Platform:** Go, Docker, K8S

### miniserve

CLI tool to serve files and dirs over HTTP.

- **Source Code:** https://github.com/svenstaro/miniserve
- **License:** MIT
- **Platform:** Rust

### ResourceSpace

Simple, fast, and free way to organise your digital assets.

- **Website:** https://www.resourcespace.com
- **Demo:** https://www.resourcespace.com/trial
- **Source Code:** https://www.resourcespace.com/svn
- **License:** BSD-4-Clause
- **Platform:** PHP

### slcl

Simple and lightweight web cloud storage.

- **Source Code:** https://codeberg.org/xavidcr/slcl
- **License:** AGPL-3.0
- **Platform:** C

### Surfer

Simple static file server with webui to manage files.

- **Source Code:** https://git.cloudron.io/cloudron/surfer
- **License:** MIT
- **Platform:** Nodejs

### TagSpaces

TagSpaces is an offline, cross-platform file manager and organiser that also can function as a note taking app. The WebDAV version of the application can be installed on top of a WebDAV servers such as Nextcloud or ownCloud.

- **Website:** https://www.tagspaces.org/
- **Demo:** https://demo.tagspaces.com
- **Source Code:** https://github.com/tagspaces/tagspaces
- **License:** AGPL-3.0
- **Platform:** Nodejs

### Tiny File Manager

Web based File Manager in PHP, simple, fast and small file manager with a single file.

- **Website:** https://tinyfilemanager.github.io
- **Demo:** https://tinyfilemanager.github.io/demo/
- **Source Code:** https://github.com/prasathmani/tinyfilemanager
- **License:** GPL-3.0
- **Platform:** PHP
