[Localization](https://en.wikipedia.org/wiki/Internationalization_and_localization) is the process of adapting code and software to other languages.

## Applications

### Accent

Developer-oriented translation tool.

- **Website:** https://www.accent.reviews/
- **Source Code:** https://github.com/mirego/accent
- **License:** BSD-3-Clause
- **Tech Stack:** Elixir/Docker

### Tolgee

Developer & translator friendly web-based localization platform enabling users to translate directly in the app they develop.

- **Website:** https://tolgee.io
- **Source Code:** https://github.com/tolgee/tolgee-platform
- **License:** Apache-2.0
- **Tech Stack:** Docker/Java

### Traduora

Translation management platform for teams.

- **Website:** https://traduora.co
- **Source Code:** https://github.com/ever-co/ever-traduora
- **License:** AGPL-3.0
- **Tech Stack:** Docker/K8S/Nodejs

### Weblate

Web-based translation tool with tight version control integration.

- **Website:** https://weblate.org
- **Source Code:** https://github.com/WeblateOrg/weblate
- **License:** GPL-3.0
- **Tech Stack:** Python/Docker/K8S
