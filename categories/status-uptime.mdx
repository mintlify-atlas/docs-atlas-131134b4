[Uptime](https://en.wikipedia.org/wiki/Uptime) is a measure of system reliability, expressed as the percentage of time a machine, typically a computer, has been working and available.

**Related:** [Monitoring](/categories/monitoring)

## Applications

### cState

Static status page for hyperfast Hugo. Clean design, minimal JS, super light HTML/CSS, high customization, optional admin panel, read-only API, IE8+. Best used with Netlify, Docker.

- **Website:** https://cstate.netlify.app/
- **Demo:** https://cstate.mnts.lt/
- **Source Code:** https://github.com/cstate/cstate
- **License:** MIT
- **Platform:** Go

### Gatus

Automated service health dashboard.

- **Website:** https://gatus.io/
- **Demo:** https://status.twin.sh
- **Source Code:** https://github.com/TwiN/gatus
- **License:** Apache-2.0
- **Platform:** Docker/K8S

### kener

Status page with incident management, easy to use and customize.

- **Website:** https://kener.ing/
- **Demo:** https://kener.ing/
- **Source Code:** https://github.com/rajnandan1/kener
- **License:** MIT

### Kuvasz Uptime

Performant, stable uptime & SSL monitoring service with brandable status pages, IAC support, Prometheus integration and a complete REST API.

- **Website:** https://kuvasz-uptime.dev
- **Demo:** https://kuvasz-uptime.dev/demo/
- **Source Code:** https://github.com/kuvasz-uptime/kuvasz
- **License:** Apache-2.0

### StatPing.ng

An easy to use Status Page for your websites and applications. Statping will automatically fetch the application and render a beautiful status page with tons of features for you to build an even better status page.

- **Website:** https://statping-ng.github.io/
- **Source Code:** https://github.com/statping-ng/statping-ng
- **License:** GPL-3.0
- **Platform:** Docker/Go
