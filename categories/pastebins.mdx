A [pastebin](https://en.wikipedia.org/wiki/Pastebin) is a type of online content-hosting service used for sharing and storing code and text.

## Applications

### bin

A paste bin that's actually minimalist.

- **Source Code:** https://github.com/w4/bin
- **License:** WTFPL/0BSD
- **Platform:** Rust

### BinPastes

Minimal pastebin supporting client-side encryption, fulltext search, one-time messages. Intended for one to few users looking for a simple pastebin deployment.

- **Source Code:** https://github.com/querwurzel/BinPastes
- **License:** Apache-2.0
- **Platform:** Java

### ByteStash

Pastebin and file storage service with a simple web interface. Supports syntax highlighting, optional user authentication and public sharing.

- **Source Code:** https://github.com/jordan-dalby/ByteStash
- **Demo:** https://github.com/jordan-dalby/ByteStash?tab=readme-ov-file#demo
- **License:** GPL-3.0
- **Platform:** Docker

### Chiyogami

Pastebin with API, client-side encryption, user accounts, syntax highlighting, markdown rendering, and more.

- **Source Code:** https://github.com/rhee876527/chiyogami
- **Demo:** https://chiyogami.myaddr.dev/
- **License:** BSD-3-Clause
- **Platform:** Docker

### dpaste

- **Website:** https://dpaste.org/
- **Source Code:** https://github.com/DarrenOfficial/dpaste
- **License:** MIT
- **Platform:** Docker/Django
