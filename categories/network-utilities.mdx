Network utilities are tools and software that help manage, monitor, and troubleshoot computer networks.

**See also:** [awesome-sysadmin/Monitoring](https://github.com/awesome-foss/awesome-sysadmin#monitoring)

## Applications

### beelzebub

Honeypot framework designed to provide a highly secure environment for detecting and analyzing cyber attacks.

- **Website:** https://beelzebub-honeypot.com/
- **Source Code:** https://github.com/mariocandela/beelzebub
- **License:** MIT
- **Platform:** Docker/K8S/Go
- **Note:** ⚠ Depends on a proprietary service outside the user's control

### Canary Tokens

Generates lightweight, embedded honeypot triggers called canary tokens for detecting unauthorized access.

- **Website:** https://canarytokens.org
- **Source Code:** https://github.com/thinkst/opencanary
- **License:** BSD-3-Clause
- **Platform:** Docker/Python

### MyIP

All in one IP Toolbox. Easy to check what's your IPs, IP geolocation, check for DNS leaks, examine WebRTC connections, speed test, ping test, MTR test, check website availability and more.

- **Website:** https://ipcheck.ing
- **Demo:** https://ipcheck.ing
- **Source Code:** https://github.com/jason5ng32/MyIP
- **License:** MIT
- **Platform:** Nodejs/Docker
- **Note:** ⚠ Depends on a proprietary service outside the user's control

### MySpeed

Speed test analysis software that shows your internet speed for up to 30 days.

- **Website:** https://myspeed.dev/
- **Source Code:** https://github.com/gnmyt/myspeed
- **License:** MIT
- **Platform:** Docker/Nodejs

### NetAlertX

Network intruder and presence detector. Scans for devices connected to your network and alerts you if new and unknown devices are found.

- **Website:** https://netalertx.com/
- **Source Code:** https://github.com/jokob-sk/NetAlertX
- **License:** GPL-3.0
- **Platform:** Docker

### Speed Test by OpenSpeedTest™

Free & Open-Source HTML5 Network Performance Estimation Tool.

- **Website:** https://openspeedtest.com/
- **Source Code:** https://github.com/openspeedtest/Speed-Test
- **License:** MIT
- **Platform:** Docker

### Speedtest Tracker

Monitor the performance and uptime of your internet connection.

- **Website:** https://docs.speedtest-tracker.dev/
- **Source Code:** https://github.com/alexjustesen/speedtest-tracker
- **License:** MIT
- **Platform:** Docker/K8S

### Upsnap

A simple Wake on LAN (WOL) dashboard app. Wake up devices on your network and see current status.

- **Source Code:** https://github.com/seriousm4x/UpSnap
- **License:** MIT
- **Platform:** Go/Docker

### Wakupator

Wake On LAN Machine Manager based on network traffic.

- **Source Code:** https://github.com/Gibus21250/Wakupator
- **License:** MIT
- **Platform:** C

### WatchYourLAN

Lightweight network IP scanner with notifications, history, export to Grafana.

- **Source Code:** https://github.com/aceberg/WatchYourLAN
- **License:** MIT
- **Platform:** Docker/Go/deb
