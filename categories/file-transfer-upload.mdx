Simplified file servers for sharing of one-time/short-lived/temporary files, providing single-click or [drag-and-drop](https://en.wikipedia.org/wiki/Drag_and_drop) upload functionality.

## Applications

### 015

A temporary file sharing platform. Focused on providing one-time, temporary file and text upload, processing, and sharing services.

- **Website:** https://send.fudaoyuan.icu
- **Source Code:** https://github.com/keven1024/015
- **License:** AGPL-3.0
- **Platform:** Docker

### Chibisafe

File uploader service that aims to to be easy to use and set up. It accepts files, photos, documents, anything you imagine and gives you back a shareable link for you to send to others.

- **Website:** https://chibisafe.app
- **Source Code:** https://github.com/chibisafe/chibisafe
- **License:** MIT
- **Platform:** Docker, Nodejs
