[Ebook](https://en.wikipedia.org/wiki/Ebook) library management software.

## Applications

### Atsumeru

Manga/comic/light novel media server with clients for Windows, Linux, macOS and Android.

- **Website:** https://atsumeru.xyz
- **Source Code:** https://github.com/Atsumeru-xyz/Atsumeru
- **Clients:** https://atsumeru.xyz/guides/#how-does-it-work
- **License:** MIT
- **Platform:** Java, Docker

### BookLogr

Manage your personal book library with ease.

- **Source Code:** https://github.com/Mozzo1000/booklogr
- **Demo:** https://demo.booklogr.app/
- **License:** Apache-2.0
- **Platform:** Docker

### BookLore

Host and manage books, with support for PDFs, eBooks, reading progress, metadata, and stats.

- **Source Code:** https://github.com/booklore-app/booklore
- **License:** GPL-3.0
- **Platform:** Docker

### Calibre Web

Browse, read and download eBooks using an existing Calibre database.

- **Source Code:** https://github.com/janeczku/calibre-web
- **License:** GPL-3.0
- **Platform:** Python

### Calibre

E-book library manager that can view, convert, and catalog e-books in most of the major e-book formats and provides a built-in Web server for remote clients.

- **Website:** https://calibre-ebook.com/
- **Demo:** https://calibre-ebook.com/demo
- **Source Code:** https://github.com/kovidgoyal/calibre
- **License:** GPL-3.0
- **Platform:** Python, deb

### Kapowarr

Build and manage a comic book library. Download, rename, move and convert issues of the volume to your liking.

- **Website:** https://casvt.github.io/Kapowarr/
- **Source Code:** https://github.com/Casvt/Kapowarr
- **License:** GPL-3.0
- **Platform:** Docker, Python

### Kavita

Cross-platform e-book/manga/comic/pdf server and web reader with user management, ratings and reviews, and metadata support.

- **Website:** https://www.kavitareader.com/
- **Demo:** https://www.kavitareader.com/#demo
- **Source Code:** https://github.com/Kareadita/Kavita
- **License:** GPL-3.0
- **Platform:** .NET, Docker

### kiwix-serve

HTTP daemon for serving wikis from ZIM files.

- **Website:** https://www.kiwix.org/en/downloads/kiwix-serve/
- **Source Code:** https://github.com/kiwix/kiwix-tools
- **License:** GPL-3.0
- **Platform:** C++

### Komga

Media server for comics/mangas/BDs with API and OPDS support, a modern web interface for exploring your libraries, as well as a web reader.

- **Website:** https://komga.org
- **Source Code:** https://github.com/gotson/komga
- **License:** MIT
- **Platform:** Java, Docker

### MyMangaDB

⚠ Manga collection manager with automatic metadata, MyAnimeList import and detailed collection statistics.

- **Source Code:** https://github.com/FabianRolfMatthiasNoll/MyMangaDB
- **License:** GPL-3.0
- **Platform:** Docker

### Stump

A fast, free and open source comics, manga and digital book server with OPDS support.

- **Website:** https://www.stumpapp.dev
- **Source Code:** https://github.com/stumpapp/stump
- **License:** MIT
- **Platform:** Rust
