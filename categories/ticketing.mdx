[Helpdesk](https://en.wikipedia.org/wiki/Help_desk_software), [bug](https://en.wikipedia.org/wiki/Bug_tracking_system) and [issue](https://en.wikipedia.org/wiki/Issue_tracking_system) tracking software to help the tracking of user requests, bugs and missing features.

**Related Categories**: Task Management & To-do Lists, Software Development - Project Management

## Applications

### Bugzilla

General-purpose bugtracker and testing tool originally developed and used by the Mozilla project.

- **Website**: [bugzilla.org](https://www.bugzilla.org/)
- **Source Code**: [github.com/bugzilla/bugzilla](https://github.com/bugzilla/bugzilla)
- **License**: MPL-2.0
- **Tech Stack**: Perl

### Frappe Helpdesk

Helpdesk software which helps you streamline your company's support, offers an easy setup, clean user interface, and automation tools to resolve customer queries efficiently.

- **Website**: [frappe.io/helpdesk](https://frappe.io/helpdesk)
- **Source Code**: [github.com/frappe/helpdesk](https://github.com/frappe/helpdesk)
- **License**: AGPL-3.0
- **Tech Stack**: Docker

### FreeScout

- **Website**: [freescout.net](https://freescout.net/)
- **Demo**: [demo.freescout.net/login](https://demo.freescout.net/login)
- **Source Code**: [github.com/freescout-help-desk/freescout](https://github.com/freescout-help-desk/freescout)
- **License**: AGPL-3.0
- **Tech Stack**: PHP, Docker

- **Website**: [glitchtip.com](https://glitchtip.com)
- **Source Code**: [gitlab.com/glitchtip/glitchtip](https://gitlab.com/glitchtip/glitchtip)
