Software for submission of [abstracts](https://en.wikipedia.org/wiki/Abstract_management) and preparation/management of academic conferences.

## Applications

### indico

Feature-rich event management system, made @ CERN, the place where the Web was born.

- **Website**: [getindico.io](https://getindico.io/)
- **Demo**: [sandbox.getindico.io](https://sandbox.getindico.io/)
- **Source Code**: [github.com/indico/indico](https://github.com/indico/indico)
- **License**: MIT
- **Tech Stack**: Python

### motion.tools (Antragsgrün)

Manage motions and amendments for (political) conventions.

- **Website**: [motion.tools](https://motion.tools/)
- **Demo**: [sandbox.motion.tools/createsite](https://sandbox.motion.tools/createsite)
- **Source Code**: [github.com/CatoTH/antragsgruen](https://github.com/CatoTH/antragsgruen)
- **License**: AGPL-3.0
- **Tech Stack**: PHP, Docker

### OpenSlides

Presentation and assembly system for managing and projecting agenda, motions and elections of an assembly.

- **Website**: [openslides.com](https://openslides.com/)
- **Demo**: [demo.os4.openslides.com/login](https://demo.os4.openslides.com/login)
- **Source Code**: [github.com/OpenSlides/OpenSlides](https://github.com/OpenSlides/OpenSlides)
- **License**: MIT
- **Tech Stack**: Docker

### osem

Event management tailored to free Software conferences.

- **Website**: [osem.io](https://osem.io/)
- **Source Code**: [github.com/openSUSE/osem](https://github.com/openSUSE/osem)
- **License**: MIT
- **Tech Stack**: Ruby, Docker

### pretalx

Web-based event management, including running a Call for Papers, reviewing submissions, and scheduling talks. Exports and imports for various related tools.

- **Website**: [pretalx.org](https://pretalx.org)
- **Source Code**: [github.com/pretalx/pretalx](https://github.com/pretalx/pretalx)
- **License**: Apache-2.0
- **Tech Stack**: Python
